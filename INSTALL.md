# How to Install MOA Calculator on Your Phone

This calculator works **completely offline** once installed. Perfect for hunting and range use!

## Installation Instructions

### For iPhone/iPad (iOS):

1. **Open Safari** (must use Safari, not Chrome)
2. Navigate to: `https://lucareynoldssa-gif.github.io/MOAcalc/`
3. Tap the **Share button** (square with arrow pointing up) at the bottom
4. Scroll down and tap **"Add to Home Screen"**
5. Tap **"Add"** in the top right corner
6. The app icon will appear on your home screen
7. **Done!** The app now works offline - no internet needed!

### For Android:

1. **Open Chrome** browser
2. Navigate to: `https://lucareynoldssa-gif.github.io/MOAcalc/`
3. Tap the **three-dot menu** in the top right
4. Tap **"Install app"** or **"Add to Home screen"**
5. Tap **"Install"** when prompted
6. The app icon will appear on your home screen
7. **Done!** The app now works offline - no internet needed!

### Alternative: Open index.html directly

If you don't want to use GitHub Pages:

1. Email yourself the files or transfer via USB
2. Save all files to your phone
3. Open the `index.html` file in your mobile browser
4. Follow the installation steps above

## Using the App

Once installed:
- Works **completely offline**
- No internet connection required
- Launch from your home screen like any other app
- Perfect for the field, range, or anywhere without signal

## Generating Icons

Before deploying, you need to create icon images:

1. Open `create-icons.html` in a web browser
2. Right-click each canvas image:
   - Save the first as `icon-192.png` (192x192 pixels)
   - Save the second as `icon-512.png` (512x512 pixels)
3. Place both PNG files in the MOAcalc folder
4. Push to GitHub

## Troubleshooting

**"Add to Home Screen" not appearing?**
- iOS: Make sure you're using Safari browser
- Android: Make sure you're using Chrome browser
- Try refreshing the page first

**App not working offline?**
- Open the app once while connected to internet
- This downloads all files for offline use
- After that, no internet needed!
